***CorpusLoadOptions*** - використовується для створення критеріїв завантаження тексту з файлу.



***CorpusClass*** - сам корпус, має поля з тренувальним і валідаційним текстом.



***CorpusLoader*** - клас для виконання методів ***Load*** i ***LoadFromText***. ***Load*** зчитує текст з файлу, а у ***LoadFromText*** розділяється на тренувальну і валідаційну частини і з нього повертається інстанс класу ***CorpusClass***.



***DefaultFileSystem*** - клас містить 2 методи: ***ReadAllText*** i ***Exists***. ***ReadAllText*** повертає текст з файлу, попередньо перевіривши, правильно вказано шлях. ***Exists*** перевіряє, чи існує файл за даним посиланням взагалі.



***CorpusSplitter*** - клас для зберігання методу ***Sрlitter***. Необхідний для методу ***LoadFromText***. Розділяє наданий текст на тренувальну і валідаційну частини у методі ***LoadFromText*** за заданими параметрами.



***CorpusTextNormalizer*** - клас з методом ***Normalize***. Зводить усе до нижнього регістру за потреби.
